# Waste Classification & Analysis System

A Python-based waste management system that:
- Classifies waste images as biodegradable or non-biodegradable using deep learning
- Visualizes waste generation trends across zones
- Provides predictive analysis for future waste loads
- Analyzes biodegradable vs non-biodegradable distribution

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the application:
```bash
python main.py
```

## Features

- Image classification (Biodegradable/Non-Biodegradable) using CNN
- Interactive data visualization with Tkinter
- Waste trend analysis across zones
- Biodegradable vs Non-Biodegradable distribution charts
- Predictive ML model for future waste estimation
